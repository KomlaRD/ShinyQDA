# ShinyQDA: R Shiny Application for Qualitative Data Analysis


![Screencast of ShinyQDA](ShinyQDA_screencast.gif)

To install, use the `remotes` package:

```
remotes::install_github('jbryer/ShinyQDA')
```

Run the Shiny app with:

```
ShinyQDA::shinyQDA()
```
